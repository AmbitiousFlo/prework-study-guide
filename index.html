HTML

  • The head element contains information about the webpage.
  • The body element represents the visable content shown to the user.
  • An HTML attribute defines additional information about an element.
  • HTML attributes are always defined in the beginning element, never the closing element.
  • An element refers to the entire element from opening tag to closing tag.
  • Tag refers only to what appears inside the angle brackets.
  • H1-H6 are elements represent the level of heading a given text block represents.
  • The p element represents a paragraph or block of text.
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items.
  • The img element contains information about images that are displayed on the webpage.
  • The src attribute is the most important because it defines the location of the image
  • The br element creates a line of empty space, or a line break, between two blocks of content.
  • The a element (which stands for anchor) creates links to the same webpage or other webpages.
  • The href attribute points to the URL for the link.

CSS

  • Cascading Style Sheets CSS is a programming language used by developers to define how webpage content is presented to users.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Use Inline CSS when you want to style an element directly in the HTML file.
  • Inline styles apply to a specific HTML tag, Useful for quick permanent changes
  • Style attribute changes colors on a webpage.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • Link element creates the connection to the external file.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location (URL) of the external resource.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • (git pull origin main) pulls the remote branch into the local main branch. When this is done your branches should be in sync.
  • Git is a version control system that lets you manage and keep track of each version of your project.
  • GitHub is a cloud-based hosting service that allows us to keep track of our projects inside repositories that are stored remotely.
  • Markdown is a lightweight plain-text formatting language. Working in a Markdown file is similar to working with a .doc, except we have to use special syntax for formatting.

JavaScript

  • JavaScript is a powerful programming language that can add interactivity to a website. It was invented by Brendan Eich.
  • Variable is simply a named container that allows us to store data in our code.
  • In JavaScript, to declare a variable, we use the (var) keyword.
  • Note that each line of JavaScript code ends with a semicolon.
  • Note that a string is contained within quotation marks.
  • You can access a whole string by refering to the variables name.
  • We can use the (console.log) method to output a message to the web console by adding an argument.
  • The word (console) refers to a test environment that developers use to check out their code.
  • .log(), is a method, which means a set of instructions that can be executed by a computer.
  • Note that whereas the style.css file is linked within the (head) element, the script.js file is linked within the (body) element.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements and loops can control the order in which code is executed, or if it is even executed at all.
  • The strict equality operator (===) checks to see if two values are equal, and returns a Boolean result true if the values are equal and false if the values are not equal.
  • An array is a single variable that is used to hold a group of data.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • Commenting out code makes it inactive without deleting it, just in case you need it later.
  • A function is simply a set of instructions that tells the computer how to perform a certain task.
